# 待办软件（To Do Software）

这是一个Python开发的待办软件，用**pyinstaller**封装，源代码使用了**tkinter**、**ttkbootstrap**、**warnings**、**datetime**和**pygame**库。

本程序自带音乐播放功能，可以自己导入。

This is a to-do software developed in Python, packaged with **pyinstaller**, and the source code uses **tkinter**, **ttkbootstrap**, **warnings**, **datetime**, and **pygame** libraries.

This program comes with music playback function, which can be imported by oneself.
