# 待办事项管理工具

## 项目简介

这是一个使用 Python 开发的桌面待办事项管理软件。软件提供了直观的任务管理界面，支持优先级分类、背景音乐播放等多种实用功能，帮助您高效管理工作与学习任务。

### 安装方式

#### 如果你是使用者

1.下载安装程序（todo_installer.exe）

2.安装软件（弹安全警告是正常的，因为搞不到签名）

3.双击使用

#### 如果你是开发者

0.下载python3.14版本

1.下载代码包

2.解压文件

2.使用以下两条指令下载所需的库

```
pip install PySide6
pip install pygame-ce
```

***tip:***

**所有用到的库：json、os、sys（python的标准库）、PySide6、pygame-ce**

**其中*pocket.py*是包，*todolist.pyw*是运行文件**

## 功能模块

### 1. 基础模块 - 任务管理核心

**核心功能：**

- **任务创建**：快速添加待办事项
- **优先级分类**：支持高、中、低三个等级
  - 高等级：红色标识，紧急重要任务
  - 中等级：黄色标识，一般任务
  - 低等级：绿色标识，次要任务
- **任务排序**：按优先级或名称排序
- **批量操作**：支持多选和批量删除
- **导入导出**：支持json格式的任务列表导入导出

### 2. 音乐模块 - 背景音乐播放

**音频功能：**

- **音乐导入**：支持 MP3 格式音频文件
- **播放控制**：开始、暂停、停止功能
- **音量调节**：可视化音量滑块控制
- **后台播放**：音乐在后台持续播放不中断任务管理
- **循环播放**：自动循环播放背景音乐

### 3. 功能模块 - 辅助工具

**实用工具：**

- **数据保存**：自动保存任务进度
- **自适应布局**：界面元素智能排列

## 使用指南

### 基本操作流程

1. **添加任务**：在文本框中输入任务内容，选择优先级，点击"创建"
2. **管理任务**：点击任务列表中的任务进行选择，支持多选
3. **设置优先级**：使用高/中/低等级按钮设置任务紧急程度
4. **播放音乐**：点击"导入音乐"选择音频文件，使用控制按钮管理播放

### 文件格式说明

使用json文件保存

```
{"task1": {"name": "任务内容", "level": "优先级"}}
示例：{"task1": {"name": "完成项目报告", "level": "高等级"}}
```

## 自定义配置

### 修改主题颜色

编辑源代码中的 `PRIORITY_INFO` 字典可自定义优先级颜色：

```python
PRIORITY_INFO = {
    '高等级': {'color': '#ffcccc', 'order': 3},
    '中等级': {'color': '#ffffcc', 'order': 2},
    '低等级': {'color': '#ccffcc', 'order': 1},
}
```
