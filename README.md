# 待办事项管理工具

一个功能全面的桌面待办事项管理应用，集任务管理、背景音乐播放和多种实用功能于一体。

## 项目简介

这是一个使用 Python 和 Tkinter 开发的桌面待办事项管理软件，已通过 PyInstaller 封装为可执行文件。软件提供了直观的任务管理界面，支持优先级分类、背景音乐播放和多种实用功能，帮助您高效管理工作与学习任务。

## 快速开始

### 系统要求

- Windows 7/8/10/11
- 4MB 可用存储空间
- 音频输出设备（用于音乐播放）

### 安装方式

#### 方式一：直接下载（推荐）

1. 下载最新版本的 `待办.exe`
2. 双击即可运行，无需安装

## 功能模块

### 1. 基础模块 - 任务管理核心

**核心功能：**

- **任务创建**：快速添加待办事项
- **优先级分类**：支持高、中、低三个等级
  - 高等级：红色标识，紧急重要任务
  - 中等级：黄色标识，一般任务
  - 低等级：绿色标识，次要任务
- **任务排序**：按优先级或名称排序
- **批量操作**：支持多选和批量删除
- **导入导出**：支持 TXT 格式的任务列表导入导出

### 2. 音乐模块 - 背景音乐播放

**音频功能：**

- **音乐导入**：支持 MP3 格式音频文件
- **播放控制**：开始、暂停、停止功能
- **音量调节**：可视化音量滑块控制
- **后台播放**：音乐在后台持续播放不中断任务管理
- **循环播放**：自动循环播放背景音乐

### 3. 功能模块 - 辅助工具

**实用工具：**

- **窗口置顶**：一键切换窗口置顶状态
- **数据保存**：自动保存任务进度
- **界面美化**：自定义背景和图标
- **启动画面**：应用启动时的欢迎界面
- **自适应布局**：界面元素智能排列

## 使用指南

### 基本操作流程

1. **添加任务**：在文本框中输入任务内容，选择优先级，点击"创建"
2. **管理任务**：点击任务列表中的任务进行选择，支持多选
3. **设置优先级**：使用高/中/低等级按钮设置任务紧急程度
4. **播放音乐**：点击"导入音乐"选择音频文件，使用控制按钮管理播放

### 文件格式说明

任务文件使用纯文本格式，每行格式：

```
任务内容------优先级
示例：完成项目报告------高等级
```

## 技术架构

### 核心技术栈

- **GUI框架**：Tkinter + ttkbootstrap（现代化界面）
- **音频处理**：Pygame.mixer（音频播放控制）
- **图像处理**：PIL/Pillow（图标和背景处理）
- **打包工具**：PyInstaller（生成独立可执行文件）

## 🎨 自定义配置

### 修改主题颜色

编辑源代码中的 `PRIORITY_INFO` 字典可自定义优先级颜色：

```python
PRIORITY_INFO = {
    '高等级': {'color': '#ffcccc', 'order': 3},
    '中等级': {'color': '#ffffcc', 'order': 2},
    '低等级': {'color': '#ccffcc', 'order': 1},
}
```

### 更换背景图片

1. 准备一张 551x395 像素的 PNG 图片
2. 重命名为 `PNG.png`
3. 替换原文件即可

## ❓ 常见问题

### Q1: 音乐文件无法导入？

A: 请检查音频文件是否为 MP3 格式，且文件路径不包含中文字符。

### Q2: 任务数据保存在哪里？

A: 任务数据默认保存在导出文件中，建议定期使用"保存"功能备份数据。

### Q3: 如何修改窗口大小？

A: 当前版本窗口大小固定，可在源代码中修改 `geometry` 参数调整。
